# Basic Usage

This section covers the fundamental features you'll use daily for monitoring system resources. Each page provides practical examples and usage patterns for different metric types.

## Available Metrics

- **[System Overview](system-overview)** - Complete system snapshot with all metrics
- **[Environment Detection](environment-detection)** - OS, kernel, architecture, virtualization, containers
- **[CPU Metrics](cpu-metrics)** - Raw CPU time counters and per-core data
- **[Memory Metrics](memory-metrics)** - Physical RAM, swap, buffers, cache
- **[Load Average](load-average)** - System load with per-core normalization
- **[Uptime](uptime)** - Boot time tracking with human-readable format
- **[Storage Metrics](storage-metrics)** - Filesystem usage and disk I/O statistics
- **[Network Metrics](network-metrics)** - Interface statistics and connection tracking

## Getting Started

Start with the **[System Overview](system-overview)** guide for a comprehensive introduction to all available metrics in a single snapshot.

For specific use cases, jump directly to the metric type you need.
